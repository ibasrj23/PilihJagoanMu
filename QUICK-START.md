# ⚡ Quick Start - Pilih Jagoan Mu

Memulai aplikasi dalam 3 langkah cepat:

## 1️⃣ Backend

```bash
cd backend
npm install
npm run dev
```

Backend berjalan di `http://localhost:5000`

## 2️⃣ Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend berjalan di `http://localhost:3000`

## 3️⃣ Open Browser

```
http://localhost:3000
```

---

**⚠️ Important:** Pastikan `.env` file sudah dibuat di backend dan `.env.local` di frontend!

Lihat `SETUP-GUIDE.md` untuk setup lengkap dengan detail.
